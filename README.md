# C# Combat RPG

A turn-based console RPG built in C# showcasing object-oriented programming.

## About

Choose your class and face off against a randomly selected computer opponent
in a round-by-round battle. Each class has unique stats, a distinct playstyle,
and its own AI logic when controlled by the computer.

## Classes

| Class   | HP  | Strength | Playstyle                          |
|---------|-----|----------|------------------------------------|
| Warrior | 180 | 20       | Tank — builds block and shield slams|
| Mage    | 70  | 25       | Glass cannon — bypasses armor      |
| Rogue   | 110 | 20       | Assassin — poisons, crits, dodges  |

## Features

- 🧙 3 playable classes with unique skill sets
- 🤖 Computer opponent with class-specific AI decision making
- 🛡️ Block, dodge, poison, and crit mechanics
- 💊 Heal skill with HP threshold restriction
- ♻️ Round-by-round combat loop with status display
